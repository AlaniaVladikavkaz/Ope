# Ветеринарная клиника API

Добро пожаловать в документацию по API ветеринарной клиники! Это API предоставляет возможность получать информацию о владельцах питомцев и их питомцах.

## Важное примечание

Это лишь пример документации для вашего API ветеринарной клиники. Реальные данные, параметры, типы данных и структура могут отличаться в зависимости от ваших потребностей.

## Эндпоинт: Получить список питомцев владельца

Этот эндпоинт позволяет получить список питомцев для конкретного владельца.

### Запрос

`GET /owners/{ownerId}/pets`

Параметры:

- `ownerId` (обязательный) - ID владельца, для которого необходимо получить список питомцев.
