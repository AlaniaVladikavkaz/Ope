# Ветеринарная клиника API

Добро пожаловать в документацию по API ветеринарной клиники! Это API предоставляет возможность получать информацию о владельцах питомцев и их питомцах.

## Эндпоинт: Получить список всех питомцев

Этот эндпоинт позволяет получить список всех питомцев, включая тех, у которых нет владельца.

### Запрос

`GET /pets`

### Пример запроса

```http
GET /pets
Ответ
Код состояния: 200 OK
Тип контента: application/json
Пример ответа:
{
  "pets": [
    {
      "id": 1,
      "name": "Барон",
      "age": 3,
      "breed": "Лабрадор",
      "owner": {
        "id": 123,
        "firstName": "Иван",
        "lastName": "Петров"
      }
    },
    {
      "id": 2,
      "name": "Лиза",
      "age": 2,
      "breed": "Бультерьер",
      "owner": null
    }
  ]
}
JSON-схемы запроса/ответа
Элемент	Множественность	Тип данных	Наименование	Пример данных
...	...	...	...	...
pets	Множественное	array	Питомцы	Пример: Список объектов питомцев (см. пример)
id	Одиночное	integer	ID питомца	Пример: 1
name	Одиночное	string	Имя питомца	Пример: "Барон"
age	Одиночное	integer	Возраст питомца	Пример: 3
breed	Одиночное	string	Порода питомца	Пример: "Лабрадор"
owner	Одиночное	object	Владелец	Пример: Объект владельца (см. пример) или null
Завершение
